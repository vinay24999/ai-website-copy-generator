# AI Website Copy Generator

## Business Chosen
Sai Krishna Hair Salon (Hyderabad)

## Objective
To generate high-converting website copy using AI prompt engineering.

## Tools Used
- ChatGPT

## What I Built
- Homepage copy
- Services content
- CTA sections

## Prompt Strategy
- Structured input (business details)
- Output divided into sections
- Focus on benefits and clarity

## Result
Ready-to-use website content for local businesses.

## Future Scope
This system can generate website copy for salons, clinics, cafes, and agencies.

This project demonstrates how AI + prompt engineering can replace expensive copywriting services for local businesses.
